## Visão Geral

O fluxo de aprovação da Motiron Chemicals é um sistema flexível e configurável que garante que todos os cadastros de materiais sejam revisados pelos responsáveis adequados antes de serem liberados para uso. O sistema automatiza notificações e acompanhamento, simplificando o processo para usuários não técnicos.

> **Importante:** O fluxo de aprovação é configurado especificamente para cada empresa. Entre em contato com o administrador para ajustar as configurações conforme suas necessidades.

## Como Funciona o Fluxo

### 1. Estrutura do Fluxo

#### Configuração Flexível

**Opções disponíveis:**
- **Fluxo Hierárquico:** Aprovações sequenciais por área
- **Fluxo Paralelo:** Aprovações simultâneas
- **Fluxo Misto:** Combinação de hierárquico e paralelo

#### Áreas de Aprovação

**Áreas comuns configuradas:**
- **Qualidade:** Controle de especificações
- **Meio Ambiente:** Conformidade ambiental
- **Segurança do Trabalho:** Riscos operacionais
- **Produção:** Viabilidade operacional
- **Compras:** Aspectos comerciais

### 2. Tipos de Usuário de Aprovação

#### Perfis de Aprovação

**Administrador Técnico (MOTIRON TECH):**
- Acesso completo ao sistema
- Configuração de fluxos
- Aprovação final de todos os materiais

**Gestor (CLIENTE):**
- Visão gerencial do fluxo
- Aprovação de materiais críticos
- Relatórios de eficiência

**Usuário de Aprovação (CLIENTE):**
- Aprovação específica por área
- Visualização de pendências
- Notificações automáticas

### 3. Estados do Fluxo

#### Status de Aprovação

| Status | Descrição | Ação Necessária |
|--------|-----------|-----------------|
| **Pendente** | Aguardando primeira aprovação | Usuário de aprovação deve revisar |
| **Em Análise** | Sendo analisado por área | Aguardar decisão |
| **Aprovado** | Aprovado por uma área | Aguardar próxima etapa |
| **Rejeitado** | Rejeitado por área | Correções necessárias |
| **Finalizado** | Aprovado por todas as áreas | Material liberado para uso |

## Processo de Aprovação

### 1. Início do Fluxo

#### Cadastro Enviado

1. **Usuário de Cadastro:** Preenche formulário completo
2. **Sistema:** Valida dados obrigatórios
3. **Sistema:** Inicia fluxo de aprovação
4. **Notificações:** Enviadas aos aprovadores

#### Validação Automática

**Verificações realizadas:**
- **Dados obrigatórios:** Todos preenchidos
- **Classificação GHS:** Aplicada automaticamente
- **FDS/FDSR:** Upload realizado
- **Responsável técnico:** Dados completos

### 2. Notificações Automáticas

#### E-mail para Aprovadores

**Conteúdo da notificação:**
- Nome do material cadastrado
- Responsável pelo cadastro
- Data e hora do cadastro
- Link direto para aprovação
- Prazo para análise (se configurado)

#### Lembretes Diários

**Configuração automática:**
- **Frequência:** Diária para pendências
- **Horário:** 9h da manhã
- **Destinatários:** Usuários de aprovação
- **Conteúdo:** Lista de pendências por área

### 3. Dashboard de Aprovação

#### Interface do Aprovador

**Funcionalidades disponíveis:**
- **Lista de Pendências:** Materiais aguardando aprovação
- **Filtros:** Por área, tipo, responsável
- **Preview:** Visualização rápida do cadastro
- **Ações:** Aprovar, Rejeitar, Solicitar Correção

#### Informações Exibidas

**Para cada material:**
- Nome e código do produto
- Responsável pelo cadastro
- Data de cadastro
- Área de responsabilidade
- Status atual no fluxo

### 4. Ações do Aprovador

#### Aprovar Material

1. **Revisar cadastro:** Verificar dados completos
2. **Validar classificação:** Confirmar GHS correto
3. **Verificar documentação:** FDS/FDSR adequada
4. **Clicar "Aprovar":** Confirmar aprovação
5. **Adicionar comentário:** Opcional

#### Rejeitar Material

1. **Identificar problema:** Dados incorretos ou incompletos
2. **Clicar "Rejeitar":** Iniciar rejeição
3. **Justificar rejeição:** Comentário obrigatório
4. **Sistema:** Notifica responsável pelo cadastro
5. **Material:** Retorna para correção

#### Solicitar Correção

1. **Identificar ajustes:** Pequenas correções necessárias
2. **Clicar "Solicitar Correção":** Manter no fluxo
3. **Adicionar comentário:** Especificar correções
4. **Sistema:** Notifica responsável pelo cadastro
5. **Material:** Aguarda correções

## Configuração do Fluxo

### 1. Configuração por Empresa

#### Áreas de Aprovação

**Configuração flexível:**
- **Quantidade:** Definida por empresa
- **Ordem:** Hierárquica ou paralela
- **Responsáveis:** Múltiplos usuários por área
- **Prazos:** Configuráveis por etapa

#### Usuários por Área

**Exemplo de configuração:**
- **Qualidade:** 2 usuários de aprovação
- **Meio Ambiente:** 1 usuário de aprovação
- **Segurança:** 1 usuário de aprovação
- **Produção:** 1 usuário de aprovação

### 2. Configuração de Prazos

#### Prazos por Etapa

**Configurações disponíveis:**
- **Prazo padrão:** 3 dias úteis
- **Prazo urgente:** 1 dia útil
- **Prazo crítico:** 4 horas
- **Sem prazo:** Aprovação quando possível

#### Alertas de Prazo

**Notificações automáticas:**
- **24h antes:** Lembrete de prazo
- **Prazo vencido:** Alerta de atraso
- **Escalação:** Notificação para gestor
- **Relatório:** Eficiência de aprovação

### 3. Configuração de Notificações

#### Tipos de Notificação

**E-mail automático:**
- **Novo cadastro:** Para aprovadores
- **Aprovação:** Para responsável pelo cadastro
- **Rejeição:** Para responsável pelo cadastro
- **Lembrete:** Para pendências diárias

#### Configuração de Destinatários

**Por tipo de material:**
- **Químicos:** Qualidade, Meio Ambiente, Segurança
- **Resíduos:** Meio Ambiente, Segurança
- **Formulações:** Qualidade, Produção

## Dicas Importantes

### Boas Práticas

#### Para Usuários de Cadastro

- **Preencha completamente:** Evite retorno para correção
- **Verifique responsáveis:** E-mails corretos
- **Mantenha FDS atualizadas:** Documentação adequada
- **Acompanhe o fluxo:** Monitore status regularmente

#### Para Usuários de Aprovação

- **Analise rapidamente:** Respeite prazos estabelecidos
- **Justifique rejeições:** Comentários claros e objetivos
- **Mantenha consistência:** Aplique critérios uniformes
- **Comunique problemas:** Alerte gestores sobre gargalos

#### Para Gestores

- **Monitore eficiência:** Acompanhe relatórios regularmente
- **Configure adequadamente:** Fluxo conforme necessidades
- **Treine a equipe:** Capacitação em procedimentos
- **Otimize processos:** Identifique e corrija gargalos

### Evite Problemas Comuns

#### Problemas de Configuração

- **Fluxo inadequado:** Configure conforme necessidades
- **Prazos irreais:** Defina prazos factíveis
- **Responsáveis incorretos:** Mantenha dados atualizados
- **Notificações não configuradas:** Configure alertas adequados

#### Problemas Operacionais

- **Aprovações lentas:** Monitore e otimize
- **Rejeições desnecessárias:** Padronize critérios
- **Falta de comunicação:** Mantenha equipe informada
- **Dados desatualizados:** Mantenha cadastros atualizados

## Erros Comuns e Soluções

### Erro: "Material não aparece na lista de pendências"
**Solução:** Verifique se o material foi enviado para aprovação e se você tem permissão para a área correspondente.

### Erro: "Não consigo aprovar o material"
**Solução:** Verifique se você tem permissão de aprovação para a área do material. Entre em contato com o administrador.

### Erro: "Notificação não recebida"
**Solução:** Verifique se o e-mail está correto nas configurações. Verifique também a pasta de spam.

### Erro: "Fluxo não configurado"
**Solução:** Entre em contato com o administrador para configurar o fluxo de aprovação adequado para sua empresa.
