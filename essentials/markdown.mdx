## Visão Geral

O cadastro de produtos químicos é o primeiro passo para gerenciar seus materiais na plataforma. Este módulo garante conformidade automática com as normas GHS e ABNT NBR 14.725:2023, simplificando o processo para usuários não técnicos.

> **Importante:** Todos os cadastros passam por fluxo de aprovação configurável. Mantenha os dados dos responsáveis sempre atualizados para agilizar o processo.

## Passo a Passo do Cadastro

### 1. Acessar o Formulário

1. Faça login na plataforma
2. Navegue para **Materiais > Produtos Químicos**
3. Clique em **"Novo Cadastro"**

### 2. Preencher Informações Básicas

#### Dados do Produto

| Campo | Obrigatório | Descrição | Exemplo |
|-------|-------------|-----------|---------|
| **Nome do Produto** | Sim | Nome comercial do produto | Ácido Sulfúrico 98% |
| **Código Interno** | Não | Código interno da empresa | AS-001 |
| **Fabricante** | Sim | Nome do fabricante | Sigma-Aldrich |
| **Número CAS** | Sim | Número CAS do componente principal | 7664-93-9 |
| **Concentração** | Sim | Concentração do produto | 98% |
| **Estado Físico** | Sim | Sólido, Líquido, Gasoso | Líquido |

#### Classificação GHS (Automática)

O sistema aplica automaticamente a classificação GHS baseada no número CAS:

- **Pictogramas de Perigo:** Atribuídos automaticamente
- **Frases H (Perigo):** Selecionadas conforme classificação
- **Frases P (Precaução):** Recomendações de segurança
- **Palavra de Advertência:** Perigo ou Atenção

:::tip
**Dica:** A classificação GHS é aplicada automaticamente, mas você pode ajustar se necessário. Sempre verifique se a classificação está correta para seu produto específico.
:::

### 3. Informações de Segurança

#### Dados de Segurança

| Campo | Obrigatório | Descrição |
|-------|-------------|-----------|
| **EPIs Necessários** | Sim | Equipamentos de proteção individual |
| **Condições de Armazenamento** | Sim | Temperatura, umidade, etc. |
| **Incompatibilidades** | Sim | Substâncias incompatíveis |
| **Procedimentos de Emergência** | Sim | Primeiros socorros, combate a incêndio |

#### Responsável Técnico

- **Nome:** Nome completo do responsável
- **E-mail:** E-mail para notificações
- **Telefone:** Contato em caso de emergência
- **Área:** Área de responsabilidade

:::warning
**Atenção:** O e-mail do responsável receberá notificações sobre o status do cadastro e lembretes de revisão de FDS.
:::

### 4. Upload de Documentação

#### Ficha de Dados de Segurança (FDS)

1. Clique em **"Upload FDS"**
2. Selecione o arquivo PDF da FDS
3. O sistema validará o formato
4. A FDS será armazenada no AWS S3

**Requisitos do arquivo:**
- Formato: PDF
- Tamanho máximo: 10MB
- Idioma: Preferencialmente português

#### Outros Documentos

- **Certificado de Análise:** Opcional
- **Especificação Técnica:** Opcional
- **Certificado de Qualidade:** Opcional

### 5. Configurações Específicas

#### Empresa Terceira (se aplicável)

Se o produto é de empresa terceira:

1. Selecione a empresa na lista
2. O sistema notificará automaticamente o responsável
3. O fluxo de aprovação será ajustado conforme configuração

#### Campos Customizados

Dependendo da configuração da sua empresa, podem aparecer campos adicionais:

- **Código de Fornecedor**
- **Categoria de Custo**
- **Local de Armazenamento**
- **Responsável de Área**

### 6. Salvar e Enviar para Aprovação

1. Revise todos os dados preenchidos
2. Clique em **"Salvar e Enviar para Aprovação"**
3. O sistema iniciará o fluxo de aprovação
4. Notificações serão enviadas aos aprovadores

## Fluxo Pós-Cadastro

### 1. Aprovação

O cadastro seguirá o fluxo configurado para sua empresa:

- **Usuários de Aprovação:** Receberão notificação por e-mail
- **Dashboard de Aprovação:** Visualização das pendências
- **Status em Tempo Real:** Acompanhamento do progresso

### 2. Notificações Automáticas

- **Responsável Técnico:** Recebe atualizações de status
- **Empresa Terceira:** Notificada sobre o cadastro
- **Áreas Envolvidas:** Meio Ambiente, Segurança do Trabalho

### 3. Após Aprovação

- **Geração de Etiquetas:** Disponível imediatamente
- **Controle de Inventário:** Produto incluído nos relatórios
- **Matriz de Incompatibilidade:** Atualizada automaticamente

## Dicas Importantes

### Boas Práticas

- **Preencha todos os campos obrigatórios:** Evita retorno para correção
- **Use nomes descritivos:** Facilita a busca e identificação
- **Mantenha FDS atualizadas:** Configure lembretes de revisão
- **Verifique responsáveis:** E-mails corretos para notificações

### Evite Problemas Comuns

- **Dados incompletos:** Preencha todos os campos marcados como obrigatórios
- **FDS desatualizada:** Use sempre a versão mais recente
- **Responsável incorreto:** Verifique nome e e-mail antes de salvar
- **Classificação GHS:** Confirme se a classificação automática está correta

### Campos Críticos

- **Número CAS:** Fundamental para classificação GHS
- **Concentração:** Afeta a classificação de perigo
- **Responsável Técnico:** Receberá todas as notificações
- **FDS:** Documento obrigatório para conformidade

## Erros Comuns e Soluções

### Erro: "Número CAS inválido"
**Solução:** Verifique se o número CAS está correto. Consulte a base de dados CAS ou entre em contato com o fabricante.

### Erro: "FDS não pode ser carregada"
**Solução:** Verifique se o arquivo é PDF e tem menos de 10MB. Tente compactar o arquivo se necessário.

### Erro: "Responsável não encontrado"
**Solução:** Verifique se o e-mail do responsável está correto e se o usuário existe no sistema.

### Erro: "Classificação GHS não aplicada"
**Solução:** Verifique se o número CAS está correto. Se o problema persistir, entre em contato com o suporte.
