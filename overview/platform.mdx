## Sobre a Motiron Chemicals v2.0

A Motiron Chemicals é uma plataforma SaaS moderna e robusta desenvolvida para gerenciamento completo de produtos químicos, resíduos perigosos e formulações internas. Nossa missão é simplificar a gestão química industrial garantindo conformidade total com as regulamentações brasileiras e internacionais.

## Arquitetura da Plataforma

### Tecnologias Utilizadas

**Frontend:**
- Next.js 14+ com TypeScript
- Tailwind CSS para estilização
- shadcn/ui para componentes
- Design responsivo e acessível

**Backend:**
- NestJS com TypeScript
- Prisma ORM para banco de dados
- JWT para autenticação
- APIs RESTful documentadas

**Infraestrutura:**
- AWS (RDS PostgreSQL, S3, CloudFront)
- Redis para cache
- Monitoramento CloudWatch
- CI/CD com GitHub Actions

### Segurança e Compliance

- **LGPD:** Proteção completa de dados pessoais
- **Criptografia:** AES-256 para dados sensíveis
- **HTTPS/TLS 1.3:** Comunicação segura
- **Auditoria:** Logs imutáveis de todas as ações
- **Backup:** Retenção de 7 anos com recuperação automática

## Módulos Principais

### 1. Autenticação e Autorização

Sistema robusto de login com 6 perfis de usuário:

- **Administrador Técnico (MOTIRON TECH):** Acesso completo ao sistema
- **Administrador (MOTIRON TECH):** Gestão de clientes e configurações
- **Gestor (CLIENTE):** Visão gerencial da empresa
- **Cadastro (CLIENTE):** Cadastro de materiais
- **Usuário de Aprovação (CLIENTE):** Aprovação de cadastros
- **Consulta (CLIENTE):** Apenas visualização

### 2. Gestão de Materiais

**Produtos Químicos:**
- Cadastro completo com classificação GHS automática
- Upload e gestão de FDS no AWS S3
- Controle de responsáveis técnicos
- Notificações automáticas por e-mail

**Resíduos Perigosos:**
- Classificação conforme NBR 10.004:2025
- Gestão de FDSR (Fichas de Dados de Segurança de Resíduos)
- Códigos IBAMA e pictogramas de transporte
- Controle de classificação Classe I e II

**Formulações Internas:**
- Cadastro de preparações internas
- Geração automática de FDS Simplificada
- Controle de datas de elaboração e revisão
- Histórico completo de alterações

### 3. Sistema de Rotulagem

**Etiquetas GHS:**
- Conformidade total com ABNT NBR 14.725:2023
- Templates personalizáveis (A4, A3, específicos)
- Pictogramas em alta resolução
- QR Code para acesso à documentação

**Etiquetas Especializadas:**
- Pequenos frascos (≤ 250ml)
- Fracionamentos com rastreabilidade
- Personalização com logomarca da empresa
- Preview antes da impressão

### 4. Inventário e Controle

## Conformidade Regulatória

### Normas Brasileiras

**ABNT NBR 14.725:2023:**
- Rotulagem de produtos químicos perigosos
- Pictogramas e frases de perigo
- Informações de EPIs necessários
- QR Code para acesso à FDS

**ABNT NBR 10.004:2025:**
- Classificação de resíduos perigosos
- Códigos IBAMA e pictogramas de transporte
- Classificação Classe I e II
- Relatórios para órgãos reguladores

### Normas Internacionais

**GHS (Globally Harmonized System):**
- Classificação e rotulagem harmonizada
- Pictogramas oficiais atualizados
- Frases H (perigo) e P (precaução)
- Sistema reconhecido mundialmente
