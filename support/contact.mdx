## Troubleshooting

### 1. Problemas de Acesso

#### Erro de Login

**Possíveis causas:**
- **Senha incorreta:** Verifique a senha
- **Usuário bloqueado:** Muitas tentativas
- **Conta inativa:** Usuário desativado
- **Problema de conexão:** Verifique internet

**Soluções:**
1. Verifique se a senha está correta
2. Use "Esqueci minha senha" se necessário
3. Entre em contato com o administrador
4. Verifique sua conexão com a internet

#### Problemas de Performance

**Sintomas:**
- **Páginas lentas:** Carregamento demorado
- **Timeouts:** Páginas não carregam
- **Erros 500:** Problemas do servidor
- **Dados não atualizados:** Cache desatualizado

**Soluções:**
1. Limpe o cache do navegador
2. Tente outro navegador
3. Verifique sua conexão
4. Entre em contato com o suporte

### 2. Problemas de Funcionalidade

#### Cadastro de Materiais

**Problemas comuns:**
- **Formulário não salva:** Dados obrigatórios faltando
- **FDS não carrega:** Arquivo muito grande
- **Classificação GHS incorreta:** Número CAS inválido
- **Responsável não encontrado:** E-mail incorreto

**Soluções:**
1. Verifique todos os campos obrigatórios
2. Compacte o arquivo PDF se necessário
3. Verifique o número CAS
4. Confirme o e-mail do responsável

#### Geração de Etiquetas

**Problemas comuns:**
- **Etiqueta não gera:** Material não aprovado
- **Pictogramas incorretos:** Classificação GHS
- **QR Code não funciona:** FDS não carregada
- **Layout inadequado:** Template incorreto

**Soluções:**
1. Verifique se o material foi aprovado
2. Confirme a classificação GHS
3. Verifique se a FDS foi carregada
4. Escolha o template correto

### 3. Problemas de Configuração

#### Fluxo de Aprovação

**Problemas comuns:**
- **Material não aparece:** Permissões incorretas
- **Notificação não recebida:** E-mail incorreto
- **Prazo não configurado:** Configuração ausente
- **Responsável não encontrado:** Usuário não cadastrado

**Soluções:**
1. Verifique suas permissões
2. Confirme o e-mail nas configurações
3. Configure prazos adequados
4. Cadastre o responsável corretamente
