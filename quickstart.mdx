## Visão Geral

A Motiron Chemicals v2.0 é uma plataforma SaaS completa para gerenciamento de produtos químicos, resíduos perigosos e formulações internas. Este guia irá ajudá-lo a configurar e começar a usar a plataforma em minutos.

> **Importante:** A plataforma garante conformidade automática com normas como ABNT NBR 14.725:2023, ABNT NBR 10.004:2025 e GHS, permitindo que você foque na operação sem se preocupar com detalhes técnicos das regulamentações.

## Passo 1: Configuração Inicial

### 1.1 Acesso à Plataforma

1. Acesse [app.motiron.com](https://app.motiron.com)
2. Use suas credenciais fornecidas pela equipe Motiron Tech
3. Na primeira vez, você será solicitado a alterar sua senha

### 1.2 Configuração da Empresa

Após o primeiro login, configure os dados da sua empresa:

- **Informações básicas:** Nome, CNPJ, endereço
- **Áreas operacionais:** Produção, Qualidade, Meio Ambiente, etc.
- **Responsáveis por área:** Nome e e-mail dos gestores
- **Configurações específicas:** Campos customizados conforme sua operação

:::tip
**Dica:** Configure todas as áreas que utilizam produtos químicos desde o início. Isso facilitará o fluxo de aprovação e controle posterior.
:::

## Passo 2: Cadastro de Materiais

### 2.1 Produtos Químicos

Comece cadastrando seus produtos químicos principais:

1. Acesse **Materiais > Produtos Químicos > Novo Cadastro**
2. Preencha os campos obrigatórios:
   - Nome do produto
   - Fabricante
   - Classificação GHS (automática)
   - Responsável técnico
   - Upload da FDS (Ficha de Dados de Segurança)

### 2.2 Resíduos Perigosos

Para resíduos perigosos:

1. Acesse **Materiais > Resíduos Perigosos > Novo Cadastro**
2. O sistema aplicará automaticamente a classificação NBR 10.004:2025
3. Faça upload da FDSR (Ficha de Dados de Segurança de Resíduos)

### 2.3 Formulações Internas

Para preparações internas:

1. Acesse **Materiais > Formulações Internas > Novo Cadastro**
2. O sistema gerará automaticamente a FDS Simplificada
3. Configure o responsável técnico e datas de revisão

:::warning
**Atenção:** Todos os cadastros passam por fluxo de aprovação configurável. Mantenha os dados dos responsáveis sempre atualizados.
:::

## Passo 3: Rotulagem e Controle

### 3.1 Geração de Etiquetas

Após a aprovação dos materiais:

1. Acesse **Rotulagem > Etiquetas GHS**
2. Selecione o material aprovado
3. Escolha o template (A4, pequenos frascos, fracionamentos)
4. Gere e imprima a etiqueta

### 3.2 Controle de Inventário

Monitore seu inventário:

1. Acesse **Inventário > Visão Geral**
2. Visualize relatórios por área
3. Exporte dados para Excel quando necessário
4. Configure lembretes de revisão de FDS

### 3.3 Segurança e Compatibilidade

Utilize os recursos de segurança:

1. **Matriz de Incompatibilidade:** Consulte antes do armazenamento
2. **Diretrizes de Armazenamento:** Siga as recomendações do sistema
3. **Alertas Automáticos:** Receba notificações de incompatibilidades

## Dicas Importantes

### Boas Práticas

- **Mantenha FDS atualizadas:** Configure lembretes de revisão
- **Use responsáveis corretos:** Facilita o fluxo de aprovação
- **Gere etiquetas sempre:** Garante conformidade regulatória
- **Monitore o inventário:** Evite produtos vencidos ou sem documentação

### Evite Problemas Comuns

- **Dados incompletos:** Preencha todos os campos obrigatórios
- **FDS desatualizadas:** Configure lembretes automáticos
- **Aprovações pendentes:** Mantenha responsáveis ativos
- **Etiquetas não geradas:** Gere sempre após aprovação
